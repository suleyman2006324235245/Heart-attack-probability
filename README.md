# Heart-attack-probability
Heart Attack Prediction Project

## About the Project
This project predicts the probability of heart attack using machine learning. The dataset is taken from Kaggle and includes different health-related health features of patients.

The model is built in a Jupyter Notebook and uses a Random Forest algorithm for prediction.

## Interface
The project also includes a simple interactive interface inside the Jupyter Notebook, where users can input values and get predictions instantly.

## What I did
- Cleaned and prepared the dataset
- Performed data analysis
- Trained a Random Forest model
- Evaluated model performance
- Built a simple input interface in the notebook

## Model Performance
- MAE: 0.0146  
- MSE: 0.0146
